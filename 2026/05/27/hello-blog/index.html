Hello, World! — 我的博客开张了

发表于2026-05-27|更新于2026-05-27|生活
|浏览量:

🎉 Hello World!

每个人的博客都有第一篇帖子，这就是我的。

为什么写博客？

记录生活，分享技术，持续成长。

过去遇到很多问题，搜索答案时总是受益于那些认真写博客的前人的分享。现在我也想把这些沉淀下来——无论是技术笔记、项目心得，还是日常的思考感悟。

博客基于

  • Hexo — 快速、简洁且高效的博客框架
  • Butterfly 主题 — 现代化的主题设计
  • GitHub Pages — 免费稳定的静态托管

接下来会写什么？

  • 💻 技术学习笔记
  • 🛠️ 项目实践经验
  • 📖 读书笔记与书评
  • 🌱 生活随想

写在最后

第一篇博客，先开张。

希望这里能成为一块安静的角落，记录成长的每一步。欢迎常来看看 ✨

— Jackie, 2026.05.27
